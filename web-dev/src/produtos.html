TELA DOS PRODUTOS

A tela de produtos será na verdade parte da navbar, como já é feito no site atualmente, mas nas nossas pesquisas notamos que as telas que explicam detalhes de cada produto são muito bagunçadas e complicadas de entender, então abraçamos a ideia de refazer o layout dessas páginas de forma que o cliente compreenda melhor qual o melhor produto para resolver os seus problemas.

Como será feito?

tópicos:

- Iremos desenvolver uma página para cada produto que eles vendem
- Reduzir os textos
- Escolher melhor as imagens para confortar o usuário
- Colocar um resumo dos produtos principais na index do portal
